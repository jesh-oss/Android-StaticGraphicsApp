# 📱 Static Graphics App

## 🚀 Overview

The Static Graphics App demonstrates drawing basic shapes (graphics primitives) such as circles, rectangles, lines, and text using Canvas in Android. It uses a custom View to render shapes programmatically.

---

## ✨ Features

* 🎨 Draw Circle, Rectangle, Line, and Text
* 🖌 Custom View using Canvas & Paint
* 📱 Lightweight and efficient rendering
* 🎯 Demonstrates Android Graphics concepts

---

## 🛠 Tech Stack

* Java
* XML
* Android Studio
* Canvas & Paint API
